   - `async def generate(self, prompt: str, job_id: str, emit: EmitFn) -> GameResult`:
     - `await emit(ProgressEvent(type="stage_start", message="Setting up project..."))`
     - `project_dir = GAMES_DIR / job_id / "project"`, `export_dir = GAMES_DIR / job_id / "export"`
     - `shutil.copytree(TEMPLATE_DIR, project_dir)` where TEMPLATE_DIR = `Path(__file__).parent.parent.parent.parent / "godot" / "templates" / "base_2d"` and GAMES_DIR = `Path(__file__).parent.parent.parent.parent / "games"`

     - `await emit(ProgressEvent(type="stage_start", message="Building for web..."))`
     - Call `await run_headless_export(project_dir, export_dir)`
     - On failure: `await emit(ProgressEvent(type="error", message="Export failed", data={"stderr": result.stderr}))`, then `await emit(None)`, then raise RuntimeError
     - On success: `await emit(ProgressEvent(type="done", message="Your game is ready."))`, then `await emit(None)`
     - Return `GameResult(job_id=job_id, wasm_path=f"/games/{job_id}/export/index.html")`

3. Update `backend/pipelines/registry.py`:
   - Import StubPipeline: `from .stub.pipeline import StubPipeline`
   - `PIPELINES: dict[str, type] = {"stub": StubPipeline}`

4. Create `backend/main.py` with:
   - `app = FastAPI(title="Moonpond")`
   - CORS middleware: allow origins `["http://localhost:3000", "http://127.0.0.1:3000"]`, methods `["GET", "POST"]`, headers `["*"]`
   - `GAMES_DIR = Path(__file__).parent.parent / "games"` ; `GAMES_DIR.mkdir(exist_ok=True)`
   - Mount static files: `app.mount("/games", StaticFiles(directory=GAMES_DIR), name="games")`
   - `POST /api/generate` endpoint:
     - Params: `req: GenerateRequest`, `background_tasks: BackgroundTasks`, `pipeline: str = "stub"`
     - Generate `job_id = str(uuid.uuid4())`
     - Create `queue: asyncio.Queue = asyncio.Queue()` (inside async handler per Pitfall 2)
     - `active_jobs[job_id] = queue`
     - Resolve pipeline: `pipeline_cls = PIPELINES.get(pipeline)`, if None return 400 error
     - Define `async def emit(event): await queue.put(event)`
     - `background_tasks.add_task(pipeline_cls().generate, req.prompt, job_id, emit)`
     - Return `GenerateResponse(job_id=job_id)`
   - `GET /api/stream/{job_id}` endpoint:
     - `async def event_generator() -> AsyncIterable[ServerSentEvent]`:
       - If job_id not in active_jobs: yield error ServerSentEvent and return
       - Loop: `event = await asyncio.wait_for(queue.get(), timeout=120)` (prevent infinite hang)
       - If event is None: `del active_jobs[job_id]`; return
       - Yield `ServerSentEvent(data=event.model_dump_json(), event=event.type)`
     - Wrap in try/except asyncio.TimeoutError to yield timeout error event
     - Return `EventSourceResponse(event_generator())`
   - IMPORTANT: The `app.mount("/games", ...)` line must come AFTER route definitions (FastAPI mounts catch all sub-paths; if mounted first, it shadows `/games` routes — but there are none here so order is technically fine; still, routes-first is defensive)

5. Update `.gitignore`: add `games/` entry if not already present (runtime output only, per CONTEXT.md decision).
