    - add_ext_resource("Script", "res://player.gd") returns a unique string ID and appears in serialize() output as [ext_resource type="Script" path="res://player.gd" id="1"]
    - add_sub_resource("RectangleShape2D", {"size": "Vector2(64, 64)"}) returns a unique ID and appears as [sub_resource] section with properties
    - add_node("Main", "Node2D", parent=None) produces root [node name="Main" type="Node2D"] with no parent= attribute
    - add_node("Player", "CharacterBody2D", parent=".") produces [node name="Player" type="CharacterBody2D" parent="."]
    - add_node with script_id= adds "script = ExtResource("N")" property line
    - add_node with instance_id= uses instance=ExtResource("N") and NO type= attribute
    - add_node with unique_name=True adds "unique_name_in_owner = true" property
    - add_node with extra_props={"shape": 'SubResource("X")'} adds property lines
    - add_connection("died", "Bird", ".", "_on_bird_died") produces [connection signal="died" from="Bird" to="." method="_on_bird_died"]
    - load_steps in header equals count of ext_resources + sub_resources
    - Sections appear in order: [gd_scene], [ext_resource]s, [sub_resource]s, [node]s, [connection]s
  </behavior>
  <action>
    Create TscnBuilder class in tscn_builder.py with:
    - __init__: empty lists for _ext_resources, _sub_resources, _nodes, _connections; _next_id=1 counter
    - add_ext_resource(res_type: str, path: str) -> str: appends (type, path, id) tuple, returns id string, increments counter
    - add_sub_resource(res_type: str, props: dict) -> str: appends (type, id, props) tuple with id like "RectangleShape2D_N", returns id
    - add_node(name, node_type, parent, *, script_id=None, instance_id=None, unique_name=False, extra_props=None): appends node dict. instance_id and node_type are mutually exclusive (assert this).
    - add_connection(signal, from_path, to_path, method): appends connection tuple
    - serialize() -> str: builds the full .tscn text string
      - Header: [gd_scene load_steps={len(ext)+len(sub)} format=3] (if load_steps=0, omit load_steps entirely)
      - ext_resources: [ext_resource type="X" path="Y" id="Z"]
      - sub_resources: [sub_resource type="X" id="Y"] followed by prop=value lines
      - nodes: [node name="X" type="Y" parent="Z"] or [node name="X" parent="Z" instance=ExtResource("N")], followed by property lines (script, unique_name_in_owner, extra_props)
      - connections: [connection signal="X" from="Y" to="Z" method="W"]
      - Blank lines between sections for readability

    - Main.tscn contains ext_resource entries for each static node's script

    - Main.tscn skips dynamic nodes (spawn_mode="dynamic")
    - Main.tscn root node is "Main" type "Node2D" with no parent
    - All scripted nodes in Main.tscn have unique_name_in_owner = true
    - Sub-scene .tscn (e.g., Bird.tscn) has root node named from scene_path stem with correct node_type
    - Sub-scene .tscn includes child nodes for each @onready %Name ref found in the .gd file
    - Sub-scene .tscn for physics bodies includes a CollisionShape2D child with RectangleShape2D sub_resource
    - Main.tscn does NOT parse @onready from main.gd (those refs are to sibling instances, not children to create)
    - No [connection] entries in Main.tscn (signals wired in _ready() per design decision)
    - Nodes with scene_path=None and NOT attached to main.gd are added inline to Main.tscn with their script
  </behavior>
  <action>
    Create scene_assembler.py with:

    1. PHYSICS_NODE_TYPES frozenset: {"CharacterBody2D", "StaticBody2D", "RigidBody2D", "Area2D"}

    2. ONREADY_PATTERN regex: r"^@onready\s+var\s+(\w+)(?:\s*:\s*(\w+))?\s*=\s*%(\w+)" with re.MULTILINE

    3. parse_onready_unique_refs(gd_source: str) -> list[dict]: extracts var_name, node_type (or None), unique_name

    4. SceneAssembler class:
       - assemble(contract: GameContract, node_files: dict[str, str]) -> dict[str, str]:
         Returns dict mapping .tscn filename -> content.

         Pass A -- Main.tscn:
         - Create TscnBuilder instance
         - Find the main script (node with scene_path matching contract.main_scene or script for the root)
         - Add root node "Main" type "Node2D"
         - For each static node in contract.nodes:
           - If node has scene_path != None and scene_path != contract.main_scene:
             add PackedScene ext_resource for scene_path, add node with instance_id, unique_name=True
           - Else if node has scene_path == contract.main_scene:
             attach script to root Main node via ext_resource
           - Else (scene_path is None):
             add Script ext_resource, add inline node with type and script_id, unique_name=True
         - No [connection] entries (per research recommendation -- signals wired in _ready())

         Pass B -- Sub-scene .tscn for each non-Main scene_path:
         - Create fresh TscnBuilder instance per file
         - Root node name = Path(scene_path).stem, type = node.node_type
         - Add Script ext_resource for node.script_path, attach to root
         - Parse @onready from the .gd file to find children
         - For each child: add_node with type (or "Node2D" if no type hint), unique_name=True
         - If root node_type is in PHYSICS_NODE_TYPES: add CollisionShape2D child with RectangleShape2D sub_resource (size Vector2(64, 64))
         - Set unique_name_in_owner on root node too
