<objective>
Convert the file generator's static system prompt to a dynamic builder that branches on perspective, add template selection to the exporter, and wire perspective through the pipeline orchestrator.

Purpose: This is the core implementation that makes 3D game generation work. The file generator prompt is the biggest change — it determines what Godot 4 3D knowledge the LLM receives. The exporter template selection ensures the correct base template (2D or 3D) is used during WASM export.

Output: Dynamic `build_generator_system_prompt()` function, perspective-aware exporter, wired pipeline, and comprehensive test coverage for both 2D and 3D paths.
</objective>

      - Set up a WorldEnvironment node with an Environment resource for ambient light and sky
