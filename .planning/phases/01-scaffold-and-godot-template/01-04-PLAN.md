Create the assets/palettes/ directory and all four Gradient .tres files. Gradient resources have a well-defined, stable text format in Godot 4 — hand-authoring is reliable for this format.

The Gradient resource stores `offsets` (array of floats 0.0-1.0) and `colors` (array of Color values in RGBA float notation). The `interpolation_mode` of 0 is linear.

**neon.tres** — Bright electric colors: magenta → cyan → yellow → hot pink
```
[gd_resource type="Gradient" load_steps=1 format=3 uid="uid://palette_neon001"]

[resource]
offsets = PackedFloat32Array(0, 0.33, 0.66, 1)
colors = PackedColorArray(1, 0, 1, 1, 0, 1, 1, 1, 1, 1, 0, 1, 1, 0, 0.251, 1)
```

Color breakdown for neon:
- 0.00: rgba(1,0,1,1) = #ff00ff magenta
- 0.33: rgba(0,1,1,1) = #00ffff cyan
- 0.66: rgba(1,1,0,1) = #ffff00 yellow
- 1.00: rgba(1,0,0.251,1) = #ff0040 hot pink

**retro.tres** — Warm arcade colors: dark red → orange → tan → cream

Create the assets/particles/ directory and all four .tscn particle scene files. These use GPUParticles2D with an inline ParticleProcessMaterial sub-resource.

The research recommends creating these in the Godot editor due to UID complexity. However, for CI purposes and to unblock Phase 2, hand-authored versions are created here. The Godot editor will regenerate UIDs on first load — this is safe behavior.

The key structure for each .tscn:
1. A `[gd_scene]` header with load_steps (1 for scene node + 1 for each sub-resource)
2. A `[sub_resource type="ParticleProcessMaterial" ...]` block with particle parameters
3. A `[node name="..." type="GPUParticles2D"]` block referencing the sub-resource

**explosion.tscn** — One-shot burst, 32 particles, wide radial spread, orange-to-red colors:
```
[gd_scene load_steps=2 format=3 uid="uid://particles_expl1"]

[sub_resource type="ParticleProcessMaterial" id="ParticleProcessMaterial_explosion"]
direction = Vector3(0, -1, 0)
spread = 180.0
initial_velocity_min = 100.0
initial_velocity_max = 300.0
gravity = Vector3(0, 98, 0)
scale_min = 4.0
scale_max = 12.0
color = Color(1, 0.5, 0, 1)

[node name="Explosion" type="GPUParticles2D"]
amount = 32
lifetime = 0.8
one_shot = true
explosiveness = 0.9
process_material = SubResource("ParticleProcessMaterial_explosion")
```

**dust.tscn** — Continuous ambient emission, slow upward drift, semi-transparent gray:
```

scale_min = 2.0
scale_max = 5.0
color = Color(0.8, 0.8, 0.8, 0.5)

[node name="Dust" type="GPUParticles2D"]
amount = 16
lifetime = 3.0
one_shot = false
```
