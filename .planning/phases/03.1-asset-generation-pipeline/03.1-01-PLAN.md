    - test_provider_registry_resolves_google: With IMAGE_PROVIDER=google, get_image_provider() returns GoogleImageProvider
    - test_provider_registry_unknown_raises: With IMAGE_PROVIDER=unknown, get_image_provider() raises ValueError

    1. Install dependencies: `cd backend && uv add openai google-genai python-dotenv`

    2. Create `backend/backend/providers/__init__.py` (empty).

    3. Create `backend/backend/providers/base.py`:
       - `ImageProvider` Protocol with `async def generate_image(self, prompt: str, width: int, height: int, style: str | None = None) -> bytes`
       - `get_image_provider() -> ImageProvider` function that reads `os.getenv("IMAGE_PROVIDER", "openai")`, lazy-imports provider classes, and instantiates
       - Raise `ValueError` for unknown provider names with helpful message listing available providers

    4. Create `backend/backend/providers/openai.py`:
       - `OpenAIImageProvider` class with `AsyncOpenAI()` client created in `__init__()` (NOT module-level)
       - `generate_image()` maps dimensions via `_map_size(width, height)` to supported sizes, calls `self._client.images.generate(model="gpt-image-1", prompt=prompt, size=size, n=1)`, returns `base64.b64decode(result.data[0].b64_json)`
       - `_map_size()`: width > height -> "1536x1024", height > width -> "1024x1536", else -> "1024x1024"

    5. Create `backend/backend/providers/google.py`:
       - `GoogleImageProvider` class with `genai.Client()` created in `__init__()`
       - `generate_image()` calls `self._client.aio.models.generate_images(model="imagen-4.0-generate-001", prompt=prompt, config=types.GenerateImagesConfig(number_of_images=1, output_mime_type="image/png"))`, returns `response.generated_images[0].image.image_bytes`
       - Width/height params accepted but Imagen 4 uses named sizes only -- all requests use default (1K)

    6. Create `backend/backend/tests/test_providers.py` with all behavior tests above. Mock `AsyncOpenAI` and `genai.Client` at the module level using `unittest.mock.patch`. Use `pytest.mark.anyio` for async tests.
